# 관리자 권한 설정 방법

새 계정을 만들면 기본값은 "일반" 권한입니다. **관리자로 만들고 싶은 계정만** 아래 과정을 따라 설정하면 됩니다.

## 1. Supabase Secret Key 확인

1. [Supabase 대시보드](https://supabase.com/dashboard) 접속 → 이 프로젝트 선택
2. 왼쪽 메뉴 **Project Settings → API**
3. **"Secret keys"** 섹션의 키 옆 눈 모양 아이콘(👁) 클릭 → 전체 키 값 확인 → 복사 아이콘으로 복사
   - (`sb_secret_...`로 시작하는 키입니다. 이 키는 절대 GitHub이나 secrets.toml에 넣지 마세요 — DB 전체를 무제한으로 조작할 수 있는 가장 강력한 키입니다.)

## 2. Supabase에서 계정 만들기 (아직 없다면)

1. 대시보드 → **Authentication → Users → Add user**
2. 이메일/비밀번호 입력 후 생성

## 3. 관리자 권한 부여

터미널(PowerShell)에서:

```
cd C:\project
python manage_users.py
```

물어보는 순서대로 입력:
1. `Supabase service_role 키를 붙여넣으세요` → 1번에서 복사한 Secret key 붙여넣기 (화면에 안 보이는 게 정상)
2. `권한을 설정할 사용자 이메일` → 관리자로 만들 계정 이메일
3. `권한 (관리자 / 일반)` → `관리자` 입력

`'이메일'  계정의 권한을 '관리자'로 설정했습니다` 메시지가 뜨면 완료입니다.

## 참고

- 아무것도 설정하지 않은 계정은 자동으로 **일반** 권한입니다.
- 일반 권한은 자재 등록/입출고 등록/메일 발송은 가능하지만, 자재 수정·삭제는 못 합니다.
- 나중에 권한을 바꾸고 싶으면(예: 일반 → 관리자, 또는 그 반대) `manage_users.py`를 다시 실행해서 같은 이메일로 다시 설정하면 됩니다.
